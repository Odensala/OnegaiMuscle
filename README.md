# Onegai Muscle
Exercise tracking app with MVVM and the latest Android architecture components. 
## Features
- Keep track of your exercise progress
- Easily search exercises
## Key components
- MVVM
- Coroutines
- Flow
- Room
- Dagger Hilt
- Unit tests
## Planned features
- Meal tracking
- Material 3
